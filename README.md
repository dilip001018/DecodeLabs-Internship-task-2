# Exploratory Data Analysis (EDA) on E-Commerce Sales Dataset

## Project Description

This project focuses on performing Exploratory Data Analysis (EDA) on an E-Commerce Sales Dataset using Microsoft Excel. The objective is to analyze customer orders, identify trends and patterns, calculate descriptive statistics, detect outliers, and generate meaningful insights from the data.

The analysis was performed using Excel features such as Pivot Tables, Charts, Filters, and Statistical Functions.

## Objectives

* Understand the structure of the dataset.
* Calculate basic statistics (Count, Mean, Median, Minimum, Maximum).
* Analyze product sales and customer purchasing behavior.
* Study payment methods and order status distribution.
* Examine coupon usage and referral sources.
* Identify trends, patterns, and outliers in the dataset.
* Present findings using charts and visualizations.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Charts and Graphs
* Descriptive Statistics

## Analysis Performed

### 1. Descriptive Statistics

* Count of Orders
* Average Order Value
* Median Order Value
* Maximum Order Value
* Minimum Order Value

### 2. Product Analysis

* Product-wise order count
* Most ordered product identification

### 3. Payment Method Analysis

* Distribution of payment methods
* Most preferred payment option

### 4. Order Status Analysis

* Delivered, Pending, Cancelled, Returned, and Shipped orders analysis

### 5. Coupon Analysis

* Most used coupon
* Least used coupon
* Orders without coupon usage

### 6. Referral Source Analysis

* Customer acquisition source analysis

### 7. Outlier Detection

* Identification of unusually high order values using visual analysis

## Key Findings

* The dataset contains customer order information from an e-commerce platform.
* Customer purchasing behavior was analyzed through product and sales trends.
* Payment preferences and order status distribution were identified.
* Coupon usage patterns were evaluated.
* Outliers were detected in TotalPrice values.
* Visualizations helped in understanding sales performance and customer behavior.

## How to Run

1. Download the dataset file.
2. Open `Dataset for Data Analytics.xlsx` using Microsoft Excel.
3. Review the dataset and perform data cleaning if required.
4. Create Pivot Tables and Charts for analysis.
5. Interpret the results and summarize key insights.
6. Refer to the project report for detailed findings.
